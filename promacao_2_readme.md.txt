O Git é um projeto de código aberto maduro e com manutenção ativa desenvolvido em 2005 por Linus Torvalds, 
o famoso criador do kernel do sistema operacional Linux. Um número impressionante de projetos de software 
depende do Git para controle de versão, incluindo projetos comerciais e de código-fonte aberto.
